## Variable Assignment

### Tuples
```python
# Empty, Single data type, multiple data type
tuple_1 = ()
tuple_2 = (1, 2, 3)
tuple_3 = (1, "Hello", 3.4)
tuple_3[0]

# String, list, tuple within tuple
tuple_4 = ("mouse", [8, 4, 6], (1, 2, 3))
tuple_4[0]
tuple_4[0][0]
```
##### Tuple Packing
```
m = 1, 2.4, "cat" # Packing
a, b, c = m  # Unpacking
```
